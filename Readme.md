# Galactic Defender

Galac Defendertic is a 2D space shooter game where you defend the galaxy from alien invaders. Dodge enemies, collect power-ups, and defeat the boss to achieve the highest score!

## Features
- **Difficulty Levels**: Choose between Easy, Normal, and Hard modes.
- **Power-Ups**: Collect shields and spread shots to enhance your abilities.
- **Boss Battles**: Face off against a challenging boss after reaching a high score.
- **Enhanced UI/UX**: Vibrant neon effects, smooth animations, improved HUD readability, and polished game over screen for an immersive experience.

## How to Play
1. **Move**: Use `W`, `A`, `S`, `D` or arrow keys to move your ship.
2. **Shoot**: Press `Space` to fire projectiles.
3. **Select Difficulty**: Choose a difficulty level on the start screen.
4. **Objective**: Destroy enemies, collect power-ups, and defeat the boss to earn points.

## Controls
- **W / A / S / D**: Move up, left, down, right.
- **Space**: Shoot.

## Installation
1. Clone the repository or download the source code.
2. Open `index.html` in your browser.
